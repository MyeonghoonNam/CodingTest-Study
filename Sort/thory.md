# 정렬
- 정렬이란 `데이터를 특정한 기준에 따라서 순서대로 나열`하는 것을 말한다.
- 정렬 알고리즘을 통해 정렬된 데이터의 경우 `이진 탐색`이 가능해지며 이진 탐색의 전처리 과정인 정렬의 개념이해가 중요하다.
- 대표적인 정렬 알고리즘으로 `버블, 선택, 삽입, 병합, 퀵 정렬` 등이 있다.
